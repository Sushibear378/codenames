#codenames
Dies ist eine Version des Spiels Codenames.
Es kann von 4 Spielern gespielt werden und muss von einem von ihnen gehostet werden. 
Die Spiellogik ist diesselbe wie bei dem gleichnamigen Brettspiel.
Instruktoren versuchen ihrem Agenten durch Hinweise zu vermitteln, welche dieser nutzt, um herrauszufinden, welche der Wörtern im 5x5 Feld zu seinem Team gehören. Diese können angeklickt werden.
