Dies ist eine digitale Version des Brettspiels Codenames auf Deutsch.
Es benötigt vier Spieler, die jeweils einen PC/Laptop in dem gleichen Netzwerk benutzen. 
Ein Spieler muss das Spiel hosten, in dem er python main.py server aufruft. Die IP dieses Rechners is die Server IP und auf dem Startbildschirm zu sehen.
Diese Server IP müssen die anderen Spieler eintragen, wenn sie das Programm durch python main.py starten. Es gibt einen Button "Standard". Wenn dieser geklickt wird, ist die Server IP automatisch die eines bestimmten Rechners im Computerraum 209.
Sobald alle 4 Spieler dem Spiel beigetreten sind, bekommen alle automatisch eine Rolle und ein Team zugewiesen und die erste Runde startet.
Der aktive Instructor muss einen Hinweis und eine Wortanzahl in die entsprechenden Felder eingeben, um seinem Mitspieler den Tip zu geben, welche Wörter dieser anklicken soll. Das Ziel des Spiels ist es, alle Wörter deines Teams (die verbleibende Anzahl kann oben auf dem BIldschirm gesehen werden) aufzudecken. Wenn man ein Wort aufdeckt, das nicht zum eigenen Team gehört, ist das andere Team am Zug. Wenn man den Spion (das schwarze Wort) aufedckt, verliert dieses Team automatisch die Runde.
Das erste Team, das 3 Runden gewinnt, gewinnt das Spiel.
